# my-AI-project

مجموعه‌ای از نوت‌بوک‌های ساده و آموزشی برای کارهای ابتدایی روی دیتاست MNIST (مناسب افراد مبتدی). وضعیت پروژه: در حال توسعه.

شامل: MNIST Classifier (طبقه‌بندی ارقام)، MNIST cGAN (تولید تصویر با Conditional GAN)، و Unsupervised MNIST (خوشه‌بندی با KMeans و کاهش بُعد).

تکنولوژی‌ها: Python 3.11 روی Windows، Jupyter Notebook/Lab، PyTorch و TensorFlow به‌همراه numpy، matplotlib، scikit-learn و tqdm.

اجرا: ابتدا وابستگی‌ها را نصب کنید: `pip install numpy matplotlib tqdm scikit-learn torch torchvision tensorflow notebook` سپس ژوپیتر را اجرا کنید: `jupyter notebook` و نوت‌بوک موردنظر را باز کرده و سلول‌ها را به ترتیب اجرا کنید. دیتاست MNIST از قبل موجود است و مسیر/نحوه استفاده داخل نوت‌بوک‌ها مشخص شده است. این ریپو ممکن است در آینده پروژه‌ها/پوشه‌های جدید اضافه کند.
