# امیر محمد خزاعی (آریان)

دانشجوی مهندسی کامپیوتر با تمرکز بر هوش مصنوعی و یادگیری ماشین.  
علاقه‌مند به تحلیل داده، پیاده‌سازی الگوریتم‌ها از پایه و طراحی سیستم‌های هوشمند.

رویکرد من مبتنی بر درک عمیق مفاهیم و پیاده‌سازی عملی آن‌هاست؛  
به‌ویژه در حوزه یادگیری بدون ناظر و الگوریتم‌های خوشه‌بندی مانند K-Means.

---

## حوزه‌های مورد علاقه پژوهشی

- یادگیری بدون ناظر (Unsupervised Learning)
- الگوریتم‌های خوشه‌بندی
- مدل‌سازی و تحلیل داده
- طراحی ایجنت‌های هوشمند و سیستم‌های تصمیم‌گیری

---

## مهارت‌های فنی

- Python

- پیاده‌سازی الگوریتم‌ها از صفر  
- تحلیل داده  
- Git

## در حال یادگیری

- بهینه‌سازی مدل‌های یادگیری ماشین  
- معماری سیستم‌های هوشمند  
- توسعه مدل‌های عمیق‌تر در TensorFlow

## رویکرد کاری

سعی می‌کنم هر مفهوم را ابتدا در سطح تحلیلی و ریاضی درک کنم و سپس آن را به‌صورت عملی پیاده‌سازی کنم.  

هدف من حرکت در مسیر پژوهش و توسعه سیستم‌های هوشمند با بنیان نظری قوی است.
